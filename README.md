# Personal Portfolio Website

A modern, high-performance, and visually stunning personal portfolio designed for software developers and creative engineers. This project showcases a blend of elegant aesthetics with state-of-the-art animations.

## ✨ Key Features

- **Premium Design System**: A curated "Nature & Tech" aesthetic using a harmonious palette of Forest Green, Bark Brown, and Sand Beige.
- **Advanced Micro-Animations**:
  - **Hero Text Reveal**: Sophisticated character-by-character entrance animation for the core headline.
  - **Magnetic Interaction**: Call-to-action buttons that subtly pull toward the cursor for a tactile feel.
  - **Interactive Card Glints**: Project and Game cards featuring dynamic light effects that track mouse movement.
  - **3D Parallax Blobs**: Smooth, multi-layered background parallax reacting to scroll and mouse movement.
- **Project Showcase**: Detailed cards for software projects and Unity-built games.
- **GitHub Integration**: Live-style statistics for repositories and contributions.
- **Full Responsiveness**: Optimized for mobile, tablet, and desktop viewing.
- **Performance Optimized**: Built with vanilla technologies and GPU-accelerated CSS for 60fps animations.

## 🛠️ Technology Stack

- **Core**: HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Vanilla CSS3
- **Typography**: Google Fonts (Playfair Display, DM Sans, DM Mono)
- **Icons**: Custom SVG icons and Emoji integrations

## 🚀 Getting Started

No installation is required. Simply open the main file in any modern web browser:

1. Clone or download the repository.
2. Open `portfolio.html` in your browser.

## 📁 Project Structure

- `portfolio.html`: The main single-page application containing all structure, styles, and logic.
- `README.md`: This documentation.
